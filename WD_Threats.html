Cyber Threats

Understanding common cyber threats is essential to staying protected online. Below is a severity chart showing how dangerous common threat types can be.

Threat Severity Chart

Threat Type Description Severity
Phishing Tricking users into giving up passwords or personal data through fake messages. High
Ransomware Malicious software that locks your files and demands payment. Critical
Social Engineering Manipulating people into revealing confidential information. Medium
Zero-Day Exploits Attacks that target unknown or unpatched software vulnerabilities. Critical
Malware Software designed to damage or disrupt systems. High
DDoS Denial-of-Service. Overloading a system with traffic making it inaccessible to the users. Medium
Trojan Disguised as real official software or files secretly containing code to execute malicious functions. High
Cryptojacking Uses a victims computer or device to secretly mine cryptocurrency. This can slow down the device drastically. High
